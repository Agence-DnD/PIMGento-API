# Compatibility with...

Akeneo
------

* PIMGento (API) is compatible with Akeneo PIM >= 2.0 (CE & EE)

Magento
--------

* PIMGento (API) is compatible with Magento following this schema:

| Magento Version    | PIMGento (API) Version  |
|--------------------|-------------------------|
| CE >= 1.9.3        | master                  |
| EE >= 1.14.3       | master                  |
